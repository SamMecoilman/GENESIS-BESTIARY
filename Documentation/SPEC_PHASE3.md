## 2. 部位破壊

### 2.1 部位定義

```
モンスター部位:
- 頭
- 胴
- 前脚（左右）
- 後脚（左右）
- 尻尾

各部位に個別のHP
```

### 2.2 破壊条件

```
部位HP = 0 で破壊

破壊効果:
- 頭: 怯みやすくなる
- 尻尾: 切断（追加剥ぎ取り）
- 脚: 転倒しやすくなる

破壊報酬:
- 部位破壊報酬（クエスト終了時）
- 尻尾は現地で剥ぎ取り可能
```

### 2.3 肉質（部位ダメージ補正）

```
例: テストモンスター
- 頭: 斬70 / 打80 / 弾60
- 胴: 斬40 / 打35 / 弾45
- 脚: 斬50 / 打45 / 弾30
- 尻尾: 斬45 / 打30 / 弾40

斬 = 切断系（大剣等）
打 = 打撃系（ハンマー等）※Phase 3では未実装
弾 = 射撃系 ※Phase 3では未実装
```

## 3. 落とし物

```
発生条件:
- 怯み時 一定確率
- 疲労時 一定確率
- 部位破壊時

アイテム:
- 光る何か（キラキラエフェクト）
- 拾う: 素材入手
- 消失: 60秒後
```

---

## 4. 罠システム

### 4.1 落とし穴

```
設置:
- 地面に設置（2秒）
- 設置後は見えにくい

効果:
- モンスターが踏むと落下
- 拘束時間: 10秒（怒り時5秒）
- 疲労時は長くなる

入手:
- 調合: トラップツール + ネット
```

### 4.2 シビレ罠

```
設置:
- 地面に設置（2秒）

効果:
- モンスターが踏むと感電
- 拘束時間: 8秒（怒り時12秒）
- 疲労時は短くなる

入手:
- 調合: トラップツール + 雷光虫
```

## 5. 投擲アイテム

### 5.1 閃光玉

```
使用:
- 投擲（前方放物線）
- 着弾で閃光発生

効果:
- モンスターを一定時間めくら状態に
- 効果時間: 15秒
- 飛行中モンスターは落下

入手:
- 調合: 素材玉 + 光蟲
```

### 5.2 音爆弾

```
使用:
- 投擲
- 着弾で大音量発生

効果:
- 潜行中モンスターを怯ませる
- 特定モンスターに効果大

入手:
- 調合: 素材玉 + 鳴き袋
```

### 5.3 ペイントボール

```
使用:
- 投擲
- モンスターに当てる

効果:
- マップにモンスター位置表示
- 効果時間: 10分

入手:
- 調合: 素材玉 + ペイントの実
```

## 6. 環境システム

### 6.1 暑さ

```
発生マップ:
- 火山、砂漠等

効果:
- 体力が徐々に減少（10秒ごとに1）
- スタミナ上限減少が早い

対策:
- クーラードリンク使用
- 効果時間: 10分
```

### 6.2 寒さ

```
発生マップ:
- 雪山等

効果:
- スタミナ回復速度低下
- スタミナ上限減少が早い

対策:
- ホットドリンク使用
- 効果時間: 10分
```

### 6.3 スタミナ上限減少

```
時間経過でスタミナ最大値が減少:
- 5分ごとに -25
- 最低値: 50

回復方法:
- 携帯食料: +25
- こんがり肉: +50
```

## 7. 肉焼き

```
素材:
- 生肉（モンスターから剥ぎ取り）

調理:
- BBQコンロ設置
- タイミングゲーム
- 音楽に合わせてボタン

結果:
- 生焼け: スタミナ+10
- こんがり肉: スタミナ+50
- 焦げ肉: スタミナ+5（煙出る）
```
