## 2. 納刀/抜刀システム

### 2.1 状態

```csharp
public enum WeaponState
{
    Sheathed,   // 納刀
    Unsheathed  // 抜刀
}
```

### 2.2 遷移

```
納刀状態:
- 移動速度: 通常（5.0 m/s）
- 攻撃: 抜刀攻撃（自動で抜刀）
- 回避: ダッシュ回避
- アイテム使用: 可能

抜刀状態:
- 移動速度: 低下（3.5 m/s）
- 攻撃: 通常攻撃
- 回避: ローリング
- アイテム使用: 不可（納刀必要）

納刀アクション:
- 発生: 1.0秒
- キャンセル不可

抜刀攻撃:
- 納刀状態から直接攻撃
- モーション値: 縦斬り + 10%
```

## 3. 回復システム

### 3.1 アイテム使用硬直

```
回復薬:
- 回復量: 30
- 使用時間: 2.0秒
- ガッツポーズ: 1.5秒（キャンセル不可）
- 合計硬直: 3.5秒

回復薬グレート:
- 回復量: 50
- 使用時間: 2.0秒
- ガッツポーズ: 1.5秒
- 合計硬直: 3.5秒

携帯食料:
- スタミナ回復: 25
- 使用時間: 1.0秒
- 硬直: 0.5秒
```

### 3.2 被弾キャンセル

```
アイテム使用中に被弾:
- 使用キャンセル
- アイテム消費なし
- のけぞり発生
```

## 4. 切れ味システム

### 4.1 ゲージ

```
切れ味レベル（低→高）:
赤 → 橙 → 黄 → 緑

補正値:
- 赤: ×0.50（弾かれやすい）
- 橙: ×0.75
- 黄: ×1.00
- 緑: ×1.125

初期武器の切れ味:
緑20 / 黄50 / 橙30 / 赤∞
```

### 4.2 消費

```
攻撃1回 = 切れ味1消費
弾かれ = 切れ味2消費

弾かれ条件:
- 肉質 × 切れ味補正 < 25
```

### 4.3 砥石

```
砥石使用:
- 使用時間: 4.0秒（砥ぎモーション）
- 回復量: 全回復
- 被弾でキャンセル（消費なし）
```

## 5. モンスター状態追加

### 5.1 怒り状態

```
発動条件:
- HP が 70% 以下になった時
- または 一定ダメージ蓄積

効果:
- 攻撃力: ×1.2
- 移動速度: ×1.3
- 攻撃頻度増加
- 見た目: 赤いオーラ（仮）

持続: 90秒
```

### 5.2 疲労状態

```
発動条件:
- 内部スタミナが 0 になった時

効果:
- 攻撃力: ×0.8
- 移動速度: ×0.7
- 攻撃失敗（よだれ垂らす）
- 見た目: 息切れモーション

持続: 60秒
回復: 時間経過 or 捕食行動
```

### 5.3 咆哮

```
発動タイミング:
- プレイヤー発見時
- 怒り移行時

効果:
- 範囲内プレイヤーにバインド
- バインド時間: 2.0秒
- バインド中は行動不可

対策（Phase 3で実装）:
- 耳栓スキル
- 回避で無効化
```

## 6. 拠点機能

### 6.1 拠点マップ

```
施設:
- クエスト受付（受付嬢）
- 加工屋（武器/防具生産）

```

```

### 6.3 加工屋

```

- 武器生産
